### Scope caveat — these numbers are partial

By default Vitest reports coverage **only for files that were imported by at least one test**. Files in `app/`, `lib/`, the Sentry config files (`sentry.{client,server,edge}.config.ts`), `instrumentation.ts`, `next.config.ts`, and any utility files not imported by a test are silently excluded from the totals above. The actual whole-repo coverage is lower. To get an honest number, set `coverage.all: true` (or an explicit `coverage.include` glob) in the Vitest config — out of scope for this snapshot.
